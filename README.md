# Мобильный шиномонтаж Москва и МО

Одностраничный сайт для мобильного шиномонтажа с PHP-обработчиком заявок.

## Структура

- `index.php`, `send.php`, `config.php` - PHP-версия для обычного хостинга.
- `assets/` - стили, JavaScript и изображения.
- `data/` - CSV-хранилище заявок для PHP-хостинга.
- `docs/` - статическая версия для GitHub Pages.

## GitHub Pages

GitHub Pages не выполняет PHP, поэтому публичная демо-версия собирается в `docs/` как статический HTML.
Формы в демо ведут на `thank-you.html`; на PHP-хостинге формы отправляются через `send.php`.

## Настройки PHP-версии

Основные настройки находятся в `config.php`: email получателя, Telegram, CSV-запись, телефон, адрес и ссылки.
